Weather App — приложение погоды
Простое веб-приложение для отображения текущей погоды в вашем городе. Использует OpenWeather API, автоматически определяет местоположение через браузер или по IP, а в случае неудачи позволяет ввести город вручную. Фон страницы меняется в зависимости от времени суток (утро, день, вечер, ночь).

Функциональность
Автоматическое определение координат через браузерную геолокацию.

Если геолокация недоступна или отклонена — определение города по IP-адресу (через ipinfo.io).

Ручной ввод названия города (резервный вариант).

Отображение текущей температуры, описания погоды, влажности и скорости ветра.

Смена фонового градиента в зависимости от времени суток (используются данные о восходе/закате из OpenWeather).

Обработка ошибок и понятные сообщения для пользователя.

Технологии
HTML5 — структура страницы.

CSS3 — стилизация, градиенты, адаптивный дизайн.

JavaScript (ES6+) — вся логика приложения.

OpenWeather API — получение данных о погоде.

ipinfo.io — геолокация по IP (требуется бесплатный токен).

Git / GitHub — контроль версий и хостинг репозитория.

Установка и запуск
1. Клонируйте репозиторий
git clone https://github.com/Dmitry-mark/weather.git
cd weather

2. Получите API-ключ OpenWeather
Зарегистрируйтесь на OpenWeather.

Перейдите в раздел API Keys.

Скопируйте ваш ключ (например, 4742acf3a7862d20268ceb26b6f6df28).

3. (Опционально) Получите токен ipinfo.io
Для работы определения местоположения по IP используется сервис ipinfo.io.

Зарегистрируйтесь на ipinfo.io.

В личном кабинете найдите ваш токен (выглядит как набор символов).

Вставьте его в файл script.js в функцию getLocationByIP:

const token = 'ВАШ_ТОКЕН'; // замените на реальный токен

Примечание: Без токена определение по IP работать не будет — приложение предложит ввести город вручную.

4. Откройте проект в браузере
Просто откройте файл index.html двойным щелчком или запустите локальный сервер.

Возможные проблемы и их решение
Ошибка геолокации браузера: пользователь должен разрешить доступ к местоположению. Если доступ запрещён, приложение автоматически переключится на IP-геолокацию.

IP-сервис не отвечает: проверьте правильность токена ipinfo.io или временные ограничения. В крайнем случае можно ввести город вручную.

OpenWeather API не работает: убедитесь, что ключ действителен и не превышен лимит запросов (бесплатный тариф — 60 запросов в минуту).
